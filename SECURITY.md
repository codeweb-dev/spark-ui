# Security Policy

Thank you for helping keep this project and its users secure.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older versions | ❌ |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report vulnerabilities using GitHub's private vulnerability
reporting feature under:

Security and quality → Advisories → Report a vulnerability

If private reporting is unavailable, contact:

allenlabrague06@gmail.com

Please include:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- Any suggested fix or mitigation

We aim to acknowledge valid reports within 3 business days and will
provide updates while the issue is investigated.

Please allow us reasonable time to investigate and release a fix before
publicly disclosing the vulnerability.
