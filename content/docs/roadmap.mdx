Components are released as they reach production quality. This roadmap tracks the larger additions currently shaping Spark UI.

## Released

- **Component registry** — A shadcn-compatible source registry with installable React components.
- **Core components** — Accessible form, layout, navigation, overlay, and feedback primitives.
- **Animated components** — Tactile Highlight, Shimmer Text, Number Ticker, Image Trail, and Logo Carousel.
- **Backgrounds** — Animated Gradient and Ghost Ether visual effects.
- **Documentation** — Live previews, source examples, installation commands, properties, and search.

## Planned

- **Showcase** — Community projects and real-world compositions built with Spark UI.
- **Templates** — Production-ready landing pages, dashboards, and application starters.
- **More components** — New primitives, blocks, and animated interactions.
- **Expanded examples** — More variants and practical usage patterns for existing components.

## Future

- **Theme presets** — Ready-made visual themes built on Spark UI design tokens.
- **Page blocks** — Larger sections composed from registry components.
- **Community submissions** — A clear workflow for sharing components, templates, and showcases.
- **Improved discovery** — Better filtering and browsing as the registry grows.

## Suggest a component

Have an idea for Spark UI? Open an issue on [GitHub](https://github.com/codeweb-dev/spark-ui) with the component, use case, and any helpful references.
