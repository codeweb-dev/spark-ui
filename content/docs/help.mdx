Need a hand with Spark UI? Choose the option below and include enough detail for us to understand what you are building.

## Found a bug?

Please include a small reproduction, the component involved, and the behavior you expected so we can investigate quickly.

[Submit a bug report](https://github.com/codeweb-dev/spark-ui/issues/new)

## Have a component suggestion?

Tell us about the component, its use case, and any helpful examples or references.

[Suggest a component](https://github.com/codeweb-dev/spark-ui/issues/new)

## Need help using Spark UI?

Share what you are trying to build, the command or code you used, and any error messages you received.

[Ask for help](https://github.com/codeweb-dev/spark-ui/issues/new)
