Keyboard renders a 60% mechanical keyboard with chunky two-tone keycaps and smooth press animations. Keys press down as you type on your physical keyboard, and clicking a key calls `onKeyPress` with its value.

## Accessibility

- Every key is a real `<button>`, so the component is fully keyboard operable with visible `focus-visible` rings.
- Icon-only keys (`⌘`, space) carry `aria-label`s.
- Press animations are disabled under `prefers-reduced-motion`; pressed keys still change appearance.
