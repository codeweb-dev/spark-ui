The Kanban component provides styled primitives for building task boards. Drag cards to reorder them smoothly or drop them into another column; the grip icon makes the interaction easy to discover while the whole card remains draggable.

## Properties

All Kanban primitives accept the standard HTML attributes for their rendered element, including `className` and native drag event handlers.

<PropsTable>
  <Prop
    name="className"
    type="string"
    default="—"
    description="Additional CSS classes for any Kanban primitive."
  />
  <Prop
    name="draggable"
    type="boolean"
    default="false"
    description="Enables native dragging when applied to KanbanCard."
  />
  <Prop
    name="data-over"
    type="boolean"
    default="false"
    description="Highlights a KanbanColumn as the active drop target."
  />
